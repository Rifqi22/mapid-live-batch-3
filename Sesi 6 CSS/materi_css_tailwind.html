1. Pengenalan CSS

CSS (Cascading Style Sheets) adalah bahasa yang digunakan untuk mengatur tampilan dan format halaman web — seperti warna, font, tata letak, dan elemen visual lainnya.

Fungsi CSS

  • Mengatur warna teks dan background
  • Mengatur ukuran font dan jenis huruf
  • Mengatur jarak (margin, padding)
  • Mengatur posisi dan layout elemen
  • Membuat website lebih rapi dan menarik

3 Cara Penulisan CSS

MetodeCaraKapan Digunakan
a. Inline CSS Atribut style="" langsung pada tag HTML Override cepat, satu elemen saja
b. Internal CSS Tag <style> di dalam <head> File HTML kecil / belajar
c. External CSS File .css terpisah, dihubungkan via <link> ✅ Proyek besar, best practice
/* a. Inline CSS — langsung di atribut style */ <section style="background-color: #dd5757; width: 500px; height: 150px"></section> /* b. Internal CSS — di dalam <style> pada <head> */ <style> section { background-color: #dd5757; width: 500px; height: 150px; } </style> /* c. External CSS — file style.css terpisah */ <link rel="stylesheet" type="text/css" href="style.css" />
⚠️ Prioritas (Cascading): Inline > Internal > External. Inline CSS akan selalu "menang" jika ada konflik. Gunakan External CSS untuk proyek nyata agar kode lebih terorganisir.

3. Selector CSS

Selector menentukan elemen mana yang akan dikenai gaya.

/* 1. Universal Selector — berlaku ke SEMUA elemen */ * { box-sizing: border-box; } /* 2. Element Selector — berdasarkan nama tag */ p { font-size: 1rem; } /* 3. Class Selector (.) — diawali titik */ .content { width: 500px; height: 100px; } /* 4. ID Selector (#) — diawali pagar, unik per halaman */ #home { background-color: #dd5757; } /* 5. Attribute Selector — berdasarkan atribut HTML */ input[type="text"] { background-color: #000000; color: #ffffff; } /* 6. Descendant Selector (spasi) — turunan elemen */ nav a { text-decoration: none; color: #ff00ff; } /* 7. Child Selector (>) — anak LANGSUNG saja */ section > h2 { text-align: center; } /* 8. Adjacent Sibling Selector (+) — elemen tepat setelah */ h1 + p { text-align: center; } /* BONUS: Pseudo-class — kondisi elemen */ a:hover { color: red; } /* saat di-hover */ li:nth-child(2) { color: blue; } /* anak ke-2 */ input:focus { border-color: blue; } /* saat aktif */ /* BONUS: Pseudo-element — bagian tertentu dari elemen */ p::first-line { font-weight: bold; } /* baris pertama */ p::before { content: "▶ "; } /* sisipkan konten sebelum */ p::after { content: " ✓"; } /* sisipkan konten setelah */

4. Box Model: Margin & Padding

Setiap elemen HTML adalah sebuah "kotak". Box Model terdiri dari 4 lapisan dari dalam ke luar:

MARGIN — jarak luar antar elemen
BORDER — garis batas elemen
PADDING — jarak dalam (konten ↔ border)
CONTENT — teks, gambar, dll.
/* Margin — jarak LUAR antar elemen */ .kotak { margin: 16px; /* semua sisi */ margin: 16px 8px; /* atas-bawah | kiri-kanan */ margin: 10px 20px 10px 20px; /* atas | kanan | bawah | kiri */ margin-top: 10px; /* sisi tertentu */ margin: 0 auto; /* center horizontal */ } /* Padding — jarak DALAM (konten ke border) */ .kotak { padding: 16px; /* semua sisi */ padding: 12px 24px; /* atas-bawah | kiri-kanan */ padding-left: 20px; /* sisi tertentu */ } /* Border — garis batas */ .kotak { border: 2px solid #1a56db; /* tebal | gaya | warna */ border-radius: 8px; /* sudut melengkung */ }

7. Positioning CSS

NilaiPenjelasanKapan Digunakan
static Default. Ikuti alur dokumen. top/left/right/bottom tidak berfungsi. Semua elemen secara default
relative Tetap di alur normal. Bisa digeser dengan top/left relatif ke posisi aslinya. Parent untuk child absolute, offset kecil
absolute Keluar dari alur. Posisi relatif ke parent terdekat yang punya position: relative/absolute/fixed. Badge, tooltip, overlay, dekorasi
fixed Keluar dari alur. Menempel di viewport, tidak ikut scroll. Navbar, tombol back-to-top, chat bubble
sticky Ikut scroll sampai batas tertentu, lalu menempel (kombinasi relative + fixed). Sidebar sticky, header tabel
.parent { position: relative; /* diperlukan agar child absolute menempel di sini */ } .child-absolute { position: absolute; top: 10px; right: 10px; } .navbar { position: fixed; /* menempel di layar, tidak ikut scroll */ top: 0; left: 0; width: 100%; z-index: 1000; /* z-index: lapisan tumpukan, makin besar = makin depan */ } .sidebar { position: sticky; /* ikut scroll sampai top: 80px, lalu menempel */ top: 80px; }
static (default) relative (geser 16px bawah+8px kanan) absolute (top:8 right:8)

8. Layout CSS — Flexbox & Grid

A. Flexbox (1 Dimensi)

Flexbox mengatur layout dalam satu arah: horizontal (row) atau vertikal (column). Cocok untuk navbar, card sejajar, centering.

.container { display: flex; /* Arah sumbu utama */ flex-direction: row; /* row (default) | column | row-reverse */ /* Distribusi ruang di sumbu utama */ justify-content: center; /* flex-start | flex-end | center | space-between | space-around */ /* Posisi di sumbu silang */ align-items: center; /* flex-start | flex-end | center | stretch */ /* Jarak antar item */ gap: 1rem; /* Wrap ke baris baru jika tidak muat */ flex-wrap: wrap; } /* Properti pada flex item */ .item { flex: 1; /* ambil sisa ruang secara merata */ flex: 0 0 200px; /* grow shrink basis — lebar tetap 200px */ }

B. CSS Grid (2 Dimensi)

Grid mengatur layout dalam dua arah sekaligus: baris (row) dan kolom (column). Cocok untuk layout halaman, dashboard, galeri.

.container { display: grid; /* Mendefinisikan kolom */ grid-template-columns: 200px 200px; /* 2 kolom fixed */ grid-template-columns: 1fr 1fr 1fr; /* 3 kolom sama lebar */ grid-template-columns: repeat(3, 1fr); /* shorthand repeat */ grid-template-columns: 2fr 1fr; /* kiri 2x lebih lebar */ grid-template-columns: repeat(auto-fit, minmax(260px, 1fr)); /* responsive otomatis! */ /* Mendefinisikan baris */ grid-template-rows: 200px 200px; /* Jarak antar sel */ gap: 1rem; column-gap: 1rem; row-gap: .5rem; } /* Item span (menggabungkan kolom/baris) */ .item-lebar { grid-column: span 2; /* menempati 2 kolom */ grid-row: span 2; /* menempati 2 baris */ }

9. Responsive Web Design

Responsive = tampilan website menyesuaikan otomatis di semua ukuran layar (Desktop | Tablet | Mobile).

Step 1: Meta Viewport (WAJIB)

<!-- Wajib ada di <head> untuk membuat website responsif --> <meta name="viewport" content="width=device-width, initial-scale=1.0">

Step 2: Media Query (Inti Responsive)

/* Breakpoint umum yang digunakan */ /* Mobile (max-width: 576px) */ @media (max-width: 576px) { body { font-size: 14px; } .card-grid { grid-template-columns: 1fr; } /* 1 kolom di HP */ } /* Tablet (max-width: 768px) */ @media (max-width: 768px) { body { background-color: lightblue; } .navbar-links { display: none; } /* sembunyikan link di tablet */ .layout { flex-direction: column; } /* flexbox jadi vertikal */ } /* Laptop (max-width: 992px) */ @media (max-width: 992px) { .sidebar { width: 240px; } } /* min-width: mobile-first approach */ @media (min-width: 768px) { .card-grid { grid-template-columns: repeat(3, 1fr); } }
NamaBreakpointPerangkat
Mobilemax-width: 576pxSmartphone
Tabletmax-width: 768pxTablet, iPad
Laptopmax-width: 992pxLaptop kecil
Desktopmax-width: 1200pxDesktop
✅ Tips Mobile First: Desain untuk layar kecil dulu tanpa media query, lalu tambahkan @media (min-width: ...) untuk layar lebih besar. Ini membuat CSS lebih ringan dan terorganisir.

CSS Framework adalah kumpulan file CSS (dan kadang JavaScript) yang menyediakan sistem layout, komponen siap pakai, dan utility class. Tujuannya mempercepat pembuatan tampilan dan menjaga konsistensi desain.

Perbandingan: Vanilla CSS vs Bootstrap vs Tailwind

AspekVanilla CSSBootstrapTailwind

Instalasi Tailwind — via CDN (Cara Paling Mudah)

<!-- Tambahkan di dalam <head> --> <script src="https://cdn.tailwindcss.com"></script> <!-- Selesai! Langsung bisa pakai class Tailwind -->
📦 Cara lain: Install via NPM (npm install tailwindcss) untuk proyek produksi. CDN cocok untuk belajar dan prototyping. Dokumentasi: tailwindcss.com/docs/installation/play-cdn

Utility Class — Cara Kerja Tailwind

Di Tailwind, styling dilakukan langsung di atribut class HTML menggunakan class-class kecil (utility). Tidak perlu file CSS terpisah!

<!-- Vanilla CSS: 2 file (HTML + CSS) --> <!-- index.html --> <button class="btn">Click Me</button> <!-- style.css --> .btn { background-color: blue; color: white; padding: 10px 16px; border-radius: 8px; } <!-- Tailwind: 1 file, class langsung di HTML --> <button class="bg-blue-500 text-white px-4 py-2 rounded-lg">Click Me</button>
💡 Cara baca class Tailwind: bg-blue-500 = bg (background) + blue (warna) + 500 (tingkat gelap). px-4 = p (padding) + x (kiri-kanan) + 4 (ukuran). Pola ini konsisten di seluruh Tailwind!

11. Materi Tambahan Penting Bonus

11.1 CSS Variables (Custom Properties) Bonus

Menyimpan nilai yang sering diulang agar mudah diubah dari satu tempat.

/* Definisikan di :root agar berlaku global */ :root { --warna-utama: #1a56db; --font-body: 'Inter', sans-serif; --radius: 10px; } /* Cara pakai: var(--nama-variabel) */ .tombol { background-color: var(--warna-utama); font-family: var(--font-body); border-radius: var(--radius); }

11.2 Transition & Transform Bonus

Membuat elemen bergerak/berubah dengan animasi halus.

.tombol { background-color: #1a56db; /* transition: property | durasi | timing-function */ transition: background-color 0.3s ease, transform 0.2s ease; } .tombol:hover { background-color: #1e3a5f; /* transform: geser, skala, rotasi */ transform: translateY(-3px); /* geser 3px ke atas */ transform: scale(1.05); /* perbesar 5% */ transform: rotate(45deg); /* putar 45 derajat */ }
🎯 Hover saya! (transition + transform)

11.3 Properti Display Bonus

.el { display: block; } /* full lebar, mulai baris baru (div, p, h1) */ .el { display: inline; } /* sebesar konten, tidak mulai baris baru (span, a) */ .el { display: inline-block; } /* seperti inline tapi bisa set width/height */ .el { display: flex; } /* flexbox container */ .el { display: grid; } /* grid container */ .el { display: none; } /* sembunyikan elemen (tidak ada di layout) */

11.4 Overflow Bonus

.el { overflow: visible; } /* konten melebihi ukuran tetap tampil (default) */ .el { overflow: hidden; } /* konten yang keluar disembunyikan */ .el { overflow: scroll; } /* selalu tampilkan scrollbar */ .el { overflow: auto; } /* scrollbar hanya muncul jika dibutuhkan */ .el { overflow-x: auto; } /* scroll horizontal saja (tabel responsif) */

11.5 Background Lengkap Bonus

.el { background-color: #1a56db; background-image: url('gambar.jpg'); background-size: cover; /* cover | contain | 100% */ background-position: center; background-repeat: no-repeat; /* Gradient */ background: linear-gradient(135deg, #1e3a5f, #1a56db); background: radial-gradient(circle, #1a56db, #1e3a5f); /* Gambar + overlay gelap (teknik umum di WebGIS hero) */ background: linear-gradient(rgba(0,0,0,0.5), rgba(0,0,0,0.5)), url('peta.jpg'); background-size: cover; }

11.6 Tips Layouting dengan CSS Tips

  1. Mulai dari Wireframe: Buat layout pakai kotak sederhana dengan warna gonjreng (merah/biru/hijau). Fokus pada struktur dulu, bukan desain.
  2. Warna Gonjreng Sementara: Tujuannya biar batas layout langsung kelihatan. Jangan takut hasilnya jelek, ini cuma tahap awal.
  3. Ganti dengan Color Palette: Setelah layout rapi, cari palette serasi di coolors.co atau colorhunt.co.
  4. Gunakan Grid & Flexbox: Flexbox → baris/kolom sederhana. Grid → layout besar 2 dimensi.
  5. Mobile First: Desain untuk HP dulu, lalu tambahkan media query untuk tablet & desktop.
  6. Tipografi: Pakai max 2 font. Hierarchy ukuran: H1 > H2 > body > caption. Line-height 1.5–1.7.
