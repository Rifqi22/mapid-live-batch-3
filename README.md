## 📚 Kurikulum & Materi Pembelajaran

Bootcamp ini dirancang secara bertahap agar rekan-rekan dapat menguasai pemrograman web untuk pemetaan (*WebGIS*) dengan struktur kurikulum sebagai berikut:

### 🌐 Sesi 1: Pengenalan HTML & Integrasi Peta Dasar
Fokus pada pemahaman fondasi halaman web dan penayangan spasial dasar:
* **Pengenalan HTML**: Sintaks dasar, struktur dokumen HTML5, serta penggunaan tag semantik.
* **Elemen Penting**: Pembuatan list, tabel data, form kontak, dan navigasi halaman.
* **Integrasi Spasial**: Memasukkan peta digital interaktif (OpenStreetMap) ke dalam halaman web menggunakan tag `iframe`.
* **📖 Materi Lengkap**: [Buka Dokumentasi HTML](file:///c:/Users/Rifqi/Work/Mapid/project/webgis%20batch%203/Live%20Session/Sesi%201/materi_html_webgis.html)
* **🛠️ Hands-on Project**: [Lihat Hasil Latihan Sesi 1](file:///c:/Users/Rifqi/Work/Mapid/project/webgis%20batch%203/Live%20Session/Sesi%201/handson_webgis.html)

### 🎨 Sesi 2: Estetika Antarmuka (CSS & Tailwind CSS)
Fokus pada memperindah tampilan visual halaman web dan menjadikannya responsif di berbagai perangkat:
* **Dasar-Dasar CSS**: Pewarnaan, tipografi modern, *box model*, serta sistem layout (*Flexbox* dan *Grid*).
* **Framework Tailwind CSS**: Mempercepat proses desain menggunakan utility-first framework CSS melalui integrasi CDN.
* **Dashboard WebGIS**: Mendesain antarmuka dashboard peta yang modern, responsif, dilengkapi sidebar, dan panel informasi.
* **📖 Materi Lengkap**: [Buka Dokumentasi CSS & Tailwind](file:///c:/Users/Rifqi/Work/Mapid/project/webgis%20batch%203/Live%20Session/Sesi%202/materi_css_tailwind.html)
* **🛠️ Hands-on CSS**: [Lihat Latihan CSS Custom](file:///c:/Users/Rifqi/Work/Mapid/project/webgis%20batch%203/Live%20Session/Sesi%202/handson_with_css.html)
* **🛠️ Hands-on Tailwind**: [Lihat Dashboard WebGIS Tailwind](file:///c:/Users/Rifqi/Work/Mapid/project/webgis%20batch%203/Live%20Session/Sesi%202/handson_with_tailwind.html)

## 💡 Tips Memulai Belajar

1. **Jalankan Secara Lokal:**
   Gunakan ekstensi **Live Server** di VS Code untuk melihat perubahan secara real-time pada browser saat memodifikasi berkas HTML/CSS.
2. **Eksplorasi Kode:**
   Jangan ragu untuk memodifikasi peta koordinat pada *iframe* atau mengganti warna tema menggunakan kelas-kelas *utility* dari Tailwind CSS di file hands-on yang telah disediakan.
3. **Gunakan Portal E-Learning:**
   Akses [Link Deploy](https://rifqi22.github.io/mapid-live-batch-3/) untuk membaca ringkasan materi secara interaktif.
